Switch Buttons Sass Mixin

A complete mixin for create simple IOS and Material Design stylish switch buttons. The html is straightforward, a simple label and a checkbox. You can use either the Sass version of just the minified, in the second case you will need to use the default classes or you can just change the classes name in the file.
